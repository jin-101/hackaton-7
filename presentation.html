AI Engine 심층 분석
🤖

Groq llama-3.3-70b 실시간 연동

돌발 이슈 텍스트 입력 → Groq API 호출 → 프레스티지·정상·할인·특가 4개 클래스 각각 가격 조정 근거 포함 추천 · ai_strategy.log 기록

🏗️ Strategy Pattern 설계
AbstractAiEngine
├─ GroqAiEngine (production)
├─ OllamaAiEngine (local)
└─ MockAiEngine (fallback)
GROQ_API_KEY 미설정 시 MockAiEngine 자동 전환
📐 EMSRb 알고리즘
Expected Marginal Seat Revenue-b
— scipy.stats.norm 기반 통계 최적화
— Logit WTP(지불의사) 구매 확률 모델
— 초기화·수정 시 자동 적용
— 클래스별 최적 보호 좌석 수 계산
⚡ AI 분석 흐름
이슈 입력
자유 텍스트 (태풍·행사·경쟁사 동향)
Groq API 호출
노선·시즌·경쟁 컨텍스트 포함 프롬프트
등급별 가격 추천
